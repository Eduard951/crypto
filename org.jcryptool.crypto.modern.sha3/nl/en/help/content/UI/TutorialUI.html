Generate hash

The hash is generated always out of the current active editor. Start the SHA-3 wizard. It can be found in Algorithms -> Hash -> SHA-3-Candidates. Chose an algorithm and the option Create Hash.

As all candidates support different bitlengths of the hash value, you have to choose one.

For better understanding you could choose the output medium of the hash value.

  • Hex editor: Displays the hash value in a hex editor.
  • Text editor: Displays the hash value blocked in a text editor, so the generated hash value is well-arranged.

By clicking on the Finish button the hash value will be generated.

Verify hash

A new hash value is generated from the editor content. This new hash value is compared with a given hash value entered in the SHA-3 wizard. The given and the new hash value and also the result of the comparison are shown in an output text editor tab.

For the verification you also have choose in the SHA-3 wizard the according algorithm and bitlength, but also the option Verify Hash via a radio button. When selecting this radio button, a new field is available to enter the given hash value.

By clicking on the Finish button the verification will be started.
